MANUAL DE USO

Sistema de pedidos online - Ivess Reggieri

1. Estructura de pantallas

  1. Paso 1 de 3: búsqueda por dirección o teléfono.
  2. Paso 2 de 3: selección de horario habilitado del cliente.
  3. Paso 3 de 3: carga de productos y comentario.
  4. Pedido enviado: confirmación del pedido.

2. Hojas que usa el sistema

No cambiar los nombres de pestañas ni los títulos de la fila 1.
  • Clientes: Nro de cliente, Dirección, Localidad, Teléfono, Activo, Lista de precios, Horario 1 a Horario 4.
  • Horarios: Nro de horario, Etiqueta, Activo.
  • ProductosPrecios: Código producto, Producto, Precio lista 1, Precio lista 2, Activo lista 1, Activo lista 2, Imagen URL.
  • Pedidos: se completa automáticamente desde la web.

3. Criterios de carga

  • Campos tipo sí/no: usar valores de lista desplegable (si o no).
  • Direcciones: guardar en texto claro, con numeración.
  • Horarios por cliente: completar columnas Horario 1, 2, 3 y 4 con números de horario existentes.
  • Lista de precios por cliente: usar 1 o 2.

4. Productos por lista

  • Si un producto debe aparecer en una lista y no en la otra, usar Activo lista 1 y Activo lista 2.
  • Ejemplo: visible solo en lista 1: Activo lista 1 = si y Activo lista 2 = no.

5. WhatsApp de asesor

  • Está disponible en todas las pantallas.
  • Número configurado: +54 9 11 3305-3315.
  • Mensaje predefinido: “Hola Ivess Reggieri, me gustaría hacer una consulta...”.

6. Cómo eliminar pedidos

  • Ir a la hoja Pedidos.
  • Seleccionar la fila completa del pedido.
  • Clic derecho → Eliminar fila.

7. URL del sistema

  • Landing inicial: /index.html.
  • App de pedidos: /pedidos.html.
  • Página base de altas: /altas.html.
