MANUAL DE USO

Sistema de pedidos online - Ivess Reggieri

1. Qué se puede editar

  • Clientes
  • Horarios
  • Asignación de horarios por cliente
  • Productos y precios

2. Hoja de cálculo (Google Sheets)

No cambiar los nombres de pestañas ni los títulos de la fila 1.
  • Clientes: alta/baja y datos de cada cliente. Campo activo: 1 habilita, 0 deshabilita.
  • En Clientes, columna lista_precio: usar 1 o 2 segun corresponda.
  • Horarios: franjas horarias disponibles. Campo activo: 1 habilita, 0 deshabilita.
  • ClienteHorarios: relación entre cliente y horarios permitidos (por ID).
  • ProductosPrecios: catálogo con precio_lista_1, precio_lista_2, activo_lista_1 y activo_lista_2.
  • Pedidos: se completa automáticamente desde la web.

3. Flujo operativo

  1. El cliente ingresa dirección o teléfono.
  2. La web identifica al cliente en la hoja Clientes.
  3. Muestra solo horarios asignados en ClienteHorarios.
  4. El cliente arma pedido y confirma.
  5. El pedido se guarda en Pedidos.

4. Cambiar precios

  1. Entrar a ProductosPrecios.
  2. Editar columna precio.
  3. Guardar: el cambio impacta en la web automáticamente.

4.1 Mostrar/ocultar por lista

  1. En ProductosPrecios, usar activo_lista_1 y activo_lista_2.
  2. Valor 1: se muestra en esa lista.
  3. Valor 0: se oculta en esa lista.

5. Alta de cliente nuevo

  1. Crear fila en Clientes con ID único.
  2. Cargar dirección y teléfono completos.
  3. Poner activo = 1.
  4. Asignar horarios en ClienteHorarios.
